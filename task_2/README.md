# Phishing Email Detection & Awareness System

## Overview

This project analyzes **phishing email samples** to identify common attack indicators and create an awareness report that helps users recognize and avoid phishing attacks.

## Objective

* Analyze phishing email examples
* Identify phishing indicators
* Classify email risk
* Explain how the attack works
* Provide prevention guidelines for users

## Tools Used

* Email Header Analyzer
  https://mxtoolbox.com/EmailHeaders.aspx
* Browser Tools (URL and domain inspection)
* Documentation (PDF report)

## Analysis Method

1. Collected phishing email samples
2. Reviewed email headers
3. Checked sender domain and links
4. Identified phishing indicators
5. Classified risk level
6. Documented findings and prevention tips
